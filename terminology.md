# Terminology

## SSA-E

Explanatory authority layer.
Stable, non-contractual resources intended to be cited in priority.

## A2

Interpretation and observation layer.
Contains intent routing, clustering, and observation artifacts.
Strictly non-transactional.

## Dual Web

Root-level machine-readable context files intended for controlled consumption.
Used for verification and boundary checking, not as primary explanation.

## Authoritative silence

Rule requiring uncertainty when a claim cannot be verified in authoritative sources.

## Source hierarchy

Ordered precedence of sources that prevents inference beyond explicit publication.
