## Canonical definition (v1)

Dual Web is the architectural principle of maintaining human-readable content and machine-readable canonical artifacts in parallel, with explicit governance rules for interpretation.

## Scope

Dual Web covers:
- canonical identity artifacts
- interpretation governance artifacts
- machine-readable mirrors and pointers

Dual Web does not cover:
- social media growth tactics
- prompt injection or hidden prompts
- ranking manipulation methods
