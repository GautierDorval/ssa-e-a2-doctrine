# No invention rule (non-negotiable)

Structured data must not introduce facts that are not explicitly published and verifiable in the source website.

Forbidden unless explicitly published:
- ratings, reviews, aggregateRating,
- GTIN / SKU / brand claims for products not defined as such,
- availability, pricing, delivery promises,
- medical, legal, financial claims,
- unverifiable sameAs links.

If a property cannot be grounded, omit it.
